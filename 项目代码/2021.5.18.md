## git指令
### commit要求
| 操作        | 指令  |
| --------   | -----:  |
| 初始化      | init  | 
| 更新        |update | 
| 修复        | fix   | 
| 删除        |remove |
| 完成      | finish  | 
| 更新        |update |
| 软撤销  |git reset --soft [commit 6] |
| 硬撤销  |git reset --hard [commit 6] |

### 分支
- 创建分支：git branch （name）
- 切换分支：git checkout （name）
- 创建并切换分支：git checkout -b (name)
- 删除分支：git branch -d (name)
- 合并分支：git merge (name)
  
### 提交
 拉取远程代码：git pull origin main

 提交本地代码：git push origin main
